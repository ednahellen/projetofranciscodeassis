# 📊 Dashboard de Gestão de Produtos

## 🚀 Visão Geral

O **Dashboard de Gestão de Produtos** é uma aplicação desktop desenvolvida em **C# com Windows Forms**, integrada ao **MySQL**, com foco em controle de estoque, organização de dados e visualização estruturada de informações.

O projeto demonstra aplicação prática de conceitos como:

* Modelagem e manipulação de banco de dados
* Operações CRUD completas
* Validações de dados
* Integração entre interface gráfica e banco de dados
* Organização lógica de código

---
